# FGM_PP

### Usage

---


1. Make workspace
    
    ```
    mkdir f1tenth_ws/src

    ```

2. Clone this repository

    ```
    cd ~/f1tenth_ws/src 
    git clone https://github.com/ikd04076/FGM_PP.git
    ```

3. Build Catkin Workspace

    ```
    cd ..
    catkin_make
    source devel/setup.bash
    ```
 
 4. Run
 
    ```
    roslaunch fgm_pp_gnu [LAUNCH_FILE]
    ```
