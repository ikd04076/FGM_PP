# FGM_PP

### Usage
---

1. Clone this repository

    ```
    cd ~/catkin_ws/src # ros workspace etc) f1tenth_ws
    git clone https://github.com/ikd04076/FGM_PP.git
    ```

2. Build Catkin Workspace

    ```
    cd ..
    catkin_make
    source devel/setup.bash
    ```
 
 3. Run
 
    ```
    roslaunch fgm_pp_gnu [LAUNCH_FILE]
    ```
